# Project: Hello World LLM

A character-level neural language model built from scratch in Python for educational purposes.
Uses PyTorch as a GPU tensor library only — training logic is all manual.
Trained on the full text of 《西游记》(Journey to the West).

## Workflow

- Always commit after meaningful changes. Don't batch unrelated changes into one commit.
- Write concise commit messages that explain *why*, not just *what*.
- Use `uv` for Python dependency management. Run with `cd src && uv run python main.py`.
